# tvelu77.github.io

## About

A portfolio website made with love and Angular !  

## How to run it ?

Clone the repository
```bash
git clone git@github.com:tvelu77/tvelu77.github.io.git
```

Install all the dependencies
```bash
npm install
```

Run it
```bash
ng serve
```
